# DSA Complete Notes

This folder contains the complete DSA notes PDF for download.

## File Required

Place the following PDF in this folder:

- **dsa-complete-notes.pdf** - Complete DSA theory notes covering all topics

## Topics Covered in the PDF

### Data Structures
1. **Arrays** - Types, Operations, Common Problems
2. **Linked List** - Singly, Doubly, Circular, Operations
3. **Stack & Queue** - Implementation, Applications
4. **Trees** - Binary Trees, BST, Traversals, Height, Diameter
5. **Graphs** - BFS, DFS, Cycle Detection, Topological Sort
6. **C++ STL** - Vectors, Pairs, Maps, Sets, Priority Queue

### Algorithms
1. **Sorting** - Bubble, Selection, Insertion, Merge, Quick, Counting
2. **Searching** - Linear, Binary, Lower/Upper Bound, 2D Matrix
3. **Recursion & Backtracking** - N-Queens, Sudoku, Permutations
4. **Dynamic Programming** - Knapsack, LCS, LIS, Grid DP, Coin Change

## How to Add the PDF

1. Create your PDF with all DSA theory content
2. Name it `dsa-complete-notes.pdf`
3. Place it in this `public/notes/` folder
4. The download button will automatically link to it at `/notes/dsa-complete-notes.pdf`
